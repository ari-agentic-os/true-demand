# 💊 付费意愿清醒剂 · true-demand  <sub>成长型创始人版</sub>

> 你的 MVP 已经拿到了第一批种子用户，但很快遇到了瓶颈：用户活跃度低？一谈收费就卡壳？不知道下一步该加功能还是换方向？
> 本技能（内部名 `true-demand`）专为解决「成长期的自嗨式调研」而生，帮你用科学的市场行为，测出种子用户账单里的真话。

面向 [Claude Code](https://claude.com/claude-code) / Agentic OS 的技能，给**已经有种子用户、但卡住了**的成长型创始人用。它强行把你的注意力从「功能好不好用（可用性）」，拉回到最常被逃避的那条线——**商业可行性：用户究竟愿不愿意为此掏钱**。

## 三大核心指令（中英镜像，文件持久化）

为了顺应成长期长达半个月甚至一个月的真实调研周期，它抛弃了「单一 Chat 窗口死磕」的模式，拆解为三个随时可单点触发、基于本地文件持久化的动作。你敲哪种语言，它就用哪种语言在本地落盘：

### 🚀 `/开局` · `/kickoff` —— 诊断现状与下发武器
- **什么时候用：** MVP 积攒了一批内测用户，但面临活跃度低、不愿付费或功能排期迷茫时。
- **它在干嘛：** ① **刺穿表象风险**——强行把注意力从「可用性」拉回最常被逃避的「商业可行性」；② **本地上下文持久化**——自动在本地创建验证看板 `看板_核心假设.md`；③ **下发大白话提纲**——吐出一份纯手工去 AI 味（经 `humanizer-zh`）、可直接拿去深聊的《提纲_真需求访谈.md》。
- **⏸️ 默认停点：** AI 在这里退场，把舞台留给你，带着提纲去跟 5 个真实用户深聊。

### 🎯 `/纠偏` · `/align` —— 访谈复盘与心智翻译
- **什么时候用：** 调研途中，觉得「用户好像在说客套话」或「心里没底」时。
- **它在干嘛：** 当你的场外顾问。你把原始笔记/录音丢进来，它帮你客观分析潜台词——比如「只要公司报销我就买」，本质上是在逃避自费，预算主体根本不是他；或者帮你发现用户愿意花大价钱买别的东西，从而撬开新的付费点；并实时修正你下一场访谈的提问姿势，防止你被假阳性信号欺骗。

### 🏁 `/确诊` · `/diagnose` —— 跨文件综合与战略转向
- **什么时候用：** 聊完 5 个真实用户，手里攒了一堆杂乱笔记时。
- **它在干嘛：** 跨文件读取 `访谈笔记/` 下所有记录，用亲和图聚类真实付费信号与高价值痛点，结合机会-解决方案树（Torres）给一份《报告_终局确诊.md》：**做**（付费路径跑通，下发排期）·**转**（现有方向是伪需求，但发现底层刚需，教你复用技术资产优雅转向）·**弃**（彻底证伪，及时止损）。

完整运作流程见 [`docs/运作流程.zh.md`](./docs/运作流程.zh.md)。

## 八大底层方法论

口语是大白话，底层依然严格执行行业黄金验证标准：

| 验证维度 | 引入的方法论 | 在工作流中的卡点动作 |
|---|---|---|
| 商业红线卡位 | IDEO 三透镜模型 | `/开局` 强行卡死 80% 权重在商业可行性，砍掉可用性自嗨 |
| 防客套测谎 | The Mom Test | 过滤掉所有未来意愿的客套话，逼出用户账单里的过去真实行为 |
| 底层动机挖掘 | JTBD 岗位任务理论 | 拆解购买背后的社会与情感痛点，诊断切换阻力 |
| 定价策略破局 | Van Westendorp PSM | 通过定性行为对比，推导出不流失内测粉的定价黄金带 |
| 信号收敛控制 | 尼尔森可用性五人定律 | 锁死 5 个真实深度定性访谈，达 85% 信号饱和，终结无休止调研 |
| 解构功能迷茫 | Teresa Torres 机会解决方案树 | 把杂乱笔记规整为「目标-机会-方案-实验」，理清功能排期 |
| 挽救活跃留存 | Superhuman 核心路径减法 | 弱水三千只取一瓢，通过超级用户反馈做功能减法，拯救低活跃度 |
| 技术资产复用 | 最小技术资产留存转向（Pivot） | 证伪后不浪费一行老代码，复用现有技术资产匹配新的高客单价场景 |

设计原理与「怎么验证这个技能本身」见 [`ABOUT.md`](./ABOUT.md)。

## 本地文件架构

```text
projects/true-demand/{日期}_{项目名}/
├── 看板_核心假设.md      # 锁定最大风险、付费可行性与核心决定
├── 提纲_真需求访谈.md    # 纯手工、去 AI 味的定性访谈脚本
├── 报告_终局确诊.md      # 亲和图聚类 + 做/转/弃确诊
└── 访谈笔记/             # 这半个月拿到的 5 份原始访谈
```

## 安装

把本仓库内容放进技能目录 `.claude/skills/true-demand/`，技能会按需读取 `references/`。中文产出经 `humanizer-zh` 去 AI 味，英文产出经 `tool-humanizer`。可选增强：`mkt-icp`、`mkt-positioning`。
